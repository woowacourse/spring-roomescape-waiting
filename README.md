## 사이클1 - 미션 (예약 대기) 기능 목록

### 1단계 - 예약 대기 신청/취소

- [x] 예약 완료된 슬롯에 대기 신청
  - [x] 순번 부여
  - [x] 같은 사용자가 같은 슬롯에 중복 대기 불가
    - [ ] 같은 사용자가 이미 예약한 슬롯에 추가로 대기 불가
    - [x] 관리자/매니저 대기 확인 및 취소
  - [x] 본인 대기 취소 가능
- [x] 예약 신청 시 이미 예약된 슬롯도 조회 가능
  
### 2단계 - 내 예약 목록 조회(상태 구분)

- [x] 예약 목록 조회 확장 (대기도 포함)
  - [x] 예약/대기 구분
  - [x] 대기 순번 표시
