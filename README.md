# 예약 대기 승인

## 1단계 - 예약 대기 승인

- [x] 예약 취소 시, 대기 1번이 자동으로 예약으로 전환된다.
  - [x] 해당 슬롯의 예약 대기가 있으면, 예약 대기를 승격한다.
    - [x] 대기 1번 예약 대기를 삭제한다.
    - [x] 새로운 예약을 저장한다.
- [x] 예약 변경 시, 대기 1번이 자동으로 예약으로 전환된다.
  - [x] 변경 전 슬롯을 기준으로 예약 대기가 있으면, 예약 대기를 승격한다.
    - [x] 대기 1번이 예약 대기를 삭제한다.
    - [x] 새로운 예약을 저장한다.
