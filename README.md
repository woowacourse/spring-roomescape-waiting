# 방탈출 미션

## 기능 요구 사항

### 1 - 예약 대기 신청/취소
- [x] 이미 다른 사용자에 의해 예약된 슬롯(날짜+시간+테마)에 대기를 신청할 수 있다.
- [x] 같은 슬롯에 대한 대기는 신청 순서대로 순번이 부여된다.
- [x] 같은 사용자가 같은 슬롯에 중복 대기할 수 없다.
- [x] 사용자는 본인의 대기를 취소할 수 있다.

### 2 - 내 예약 목록 조회 (상태 구분)
- 내 예약 목록 조회
    - [x] 사용자의 예약과 대기가 상태로 구분되어 함께 표시된다.
    - [x] 대기에는 본인의 대기 순번도 함께 보여준다.

### 3 - 예약 대기 승인
- [ ] 예약 대기를 승인하여 예약으로 전환한다.
  - 예약 취소 시 대기 1번이 자동으로 예약 전환된다.
- [ ] 대기가 예약으로 전환되면 해당 슬롯의 나머지 대기 순번이 재정렬된다.
- [ ] 예약이 취소되면 해당 슬롯의 대기 순번이 재정렬된다.

### 4 - 예외 처리
- [x] 지난 날짜•시간에 대한 예약 생성은 불가하다.
- [x] 같은 날짜•시간•테마에는 하나의 확정 예약만 가능하다.
- [x] 같은 사용자가 중복으로 예약•예약 대기할 수 없다.
- [x] 예약이 존재하는 시간•테마는 삭제할 수 없다.
- [x] 유효하지 않은 입력값(빈 이름, 잘못된 날짜 형식 등)을 거부한다.
- [x] 변경하려는 날짜•시간에 이미 예약•예약 대기가 있으면 변경할 수 없다.

## API 명세

### 사용자 API

| 분류 | 기능 | Method | URL | 성공 응답 | 에러 응답 |
|-------|--------------|----------|-------------------------------------------------------|------------------|---------------------------------------------------------------------|
| 예약 | 예약 생성 | `POST` | `/reservations` | `200 OK` | `400 Bad Request`, `404 Not Found`, `409 Conflict` |
| 예약 | 예약 조회 | `GET` | `/reservations` | `200 OK` | - |
| 예약 | 내 예약 조회 | `GET` | `/reservations/my-reservation?name={name}` | `200 OK` | - |
| 예약 | 예약 변경 | `PATCH` | `/reservations/{id}` | `200 OK` | `400 Bad Request`, `404 Not Found`, `409 Conflict` |
| 예약 | 예약 삭제 | `DELETE` | `/reservations/{id}` | `204 No Content` | `404 Not Found` |
| 예약 시간 | 예약 시간 목록 조회 | `GET` | `/times` | `200 OK` | - |
| 예약 시간 | 예약 가능한 시간 조회 | `GET` | `/times/available-times?themeId={themeId}&date={date}` | `200 OK` | `400 Bad Request`, `404 Not Found` |
| 테마 | 테마 목록 조회 | `GET` | `/themes` | `200 OK` | - |
| 테마 | 인기 테마 조회 | `GET` | `/themes/popular?limit={limit}` | `200 OK` | - |

### 관리자 API

| 분류 | 기능 | Method | URL | 성공 응답 | 에러 응답 |
|-------|----------|----------|-----------------------|------------------|-----------------------------------|
| 예약 시간 | 예약 시간 생성 | `POST` | `/admin/times` | `200 OK` | `400 Bad Request`, `409 Conflict` |
| 예약 시간 | 예약 시간 삭제 | `DELETE` | `/admin/times/{id}` | `204 No Content` | `404 Not Found`, `409 Conflict` |
| 테마 | 테마 생성 | `POST` | `/admin/themes` | `201 Created` | `400 Bad Request`, `409 Conflict` |
| 테마 | 테마 삭제 | `DELETE` | `/admin/themes/{id}` | `204 No Content` | `404 Not Found`, `409 Conflict` |
