# 방탈출 예약 관리

## 🙋기능 요구사항

### 관리자 예약 관리

- 관리자는 예약을 예약자 이름, 예약 날짜, 예약 시간, 테마 정보를 입력해 예약을 추가할 수 있다
- 관리자는 전체 예약 목록을 조회할 수 있다
- 관리자는 특정 예약을 삭제할 수 있어야 한다.

### 관리자 예약 시간 관리

- 관리자는 시간을 입력해서 예약 시간을 등록할 수 있다.
- 관리자는 전체 예약 시간 목록을 조회할 수 있다.
- 관리자는 예약 시간을 삭제할 수 있다.

### 관리자 테마 관리

- 관리자는 테마의 이름, 설명, 썸네일 이미지 URL를 입력하여 등록할 수 있다.
- 관리자는 전체 테마 목록을 조회할 수 있다.
- 관리자는 특정 테마를 삭제할 수 있다.

### 사용자 예약

- 사용자는 전체 예약 시간 목록에서 시간을 선택할 수 있다.
- 사용지는 예약자 이름, 예약 날짜, 예약 시간, 테마 정보를 입력해서 예약을 할 수 있다.
- 사용자는 자신의 이름으로 본인의 예약 목록을 조회할 수 있다.
- 사용자는 자신의 예약의 날짜, 시간을 변경할 수 있다. (테마는 X)
- 사용자는 자신의 예약을 취소할 수 있다.

### 사용자 예약 대기
- 사용자는 이미 다른 사용자에 의해 예약된 슬롯(날짜+시간+테마)에 대기를 신청할 수 있다.
- 같은 슬롯에 대한 대기는 신청 순서대로 순번이 부여된다
- 예약이 변경/취소되면 대기 중인 예약 중 제일 빠른 예약이 확정된다.

### 인기 테마

- 사용자는 최근 1주일 동안 예약이 많았던 테마 상위 10개를 확인할 수 있어야 한다.

## 📜 서비스 정책

### 예약

- 지나간 날짜, 시간에 대한 예약 생성/변경/삭제는 불가능하다
- 같은 날짜, 시간, 테마에 확정 예약이 있으면 대기로 등록된다.
- 사용자는 다른 사람의 예약을 변경/취소할 수 없다.
- 이미 취소된 예약은 변경/취소할 수 없다.

### 테마

- 모든 테마의 시작 시간과 소요 시간은 동일하다.
- 예약이 존재하는 테마는 삭제할 수 없다.
- 사용자는 테마를 추가/삭제할 수 없다.

### 예약 시간

- 예약이 존재하는 시간은 삭제할 수 없다.
- 사용자는 예약 시간을 추가/삭제할 수 없다.

### 예약 대기

- 같은 사용자가 같은 슬롯에 중복 대기할 수 없다.
- 사용자는 본인의 대기를 취소할 수 있다.

## 🌐 API 명세

### 🔐 관리자 API

#### 예약

##### 관리자 예약 목록 조회 API ✅

**Request**

```http
GET /admin/reservations HTTP/1.1
```

**Response**

```http
HTTP/1.1 200
Content-Type: application/json

{
  "reservations": [
    {
      "id": 1,
      "guestName": "브라운",
      "date": "2023-08-05",
      "time": {
        "id": 1,
        "startAt": "10:00"
      },
      "theme": {
        "id": 1,
        "name": "레벨2 탈출",
        "description": "우테코 레벨2를 탈출하는 내용입니다.",
        "thumbnail": "https://i.pinimg.com/236x/6e/bc/46/6ebc461a94a49f9ea3b8bbe2204145d4.jpg"
      }
    }
  ]
}
```

##### 관리자 예약 삭제 API ✅

**Request**

```http
DELETE /admin/reservations/1 HTTP/1.1
```

**Response**

```http
HTTP/1.1 204
```

에러 상황:

| 에러 상황               | 상태코드  | 구현여부 |
|---------------------|-------|------|
| 유효하지 않는 입력값         | `400` | ✅    |
| 예약이 존재하지 않을 때       | `404` | ✅    |
| 이미 취소된 예약인 경우       | `409` | ✅     |
| 이미 시작된 예약을 취소하려는 경우 | `422` | ✅     |

## 🖥️ 프론트엔드 페이지

### 공통

- 상단 전환 버튼으로 사용자 페이지와 관리자 페이지를 이동할 수 있다.

### 사용자 예약 페이지

- 경로: `http://localhost:8080`
- 최근 1주 기준 인기 테마 상위 10개를 확인하고, 인기 테마를 눌러 바로 선택할 수 있다.
- 날짜, 테마, 등록된 전체 예약 시간을 선택할 수 있다.
- 예약 정보 요약에서 선택한 날짜, 테마, 시간, 신청 상태를 확인할 수 있다.
- 예약자 이름을 입력해 예약/대기 신청을 할 수 있다.
- 빈 슬롯은 확정 예약으로 등록되고, 이미 확정 예약이 있는 슬롯은 대기 예약으로 등록된다.
- 대기 신청이 완료되면 대기 순번을 메시지와 토스트로 확인할 수 있다.
- 예약자 이름으로 본인의 예약 목록을 조회할 수 있다.
- 조회된 예약은 `확정`, `대기 n번`, `취소` 상태 배지로 구분된다.
- 본인의 예약에 한해 날짜와 시간을 수정할 수 있다.
- 수정 시에도 등록된 전체 예약 시간을 선택할 수 있고, 확정 예약이 있는 슬롯으로 변경하면 대기로 전환된다.
- 본인의 예약에 한해 예약을 취소할 수 있다.
- 예약 취소 시 조회용 이름과 별도로 예약자 이름을 다시 입력한다.
- 취소된 예약은 목록에 취소 상태로 남고 수정/취소 버튼은 표시되지 않는다.

### 관리자 페이지

- 경로: `http://localhost:8080/admin`
- 예약자 이름, 날짜, 테마, 등록된 전체 예약 시간을 입력해 예약이나 대기를 등록할 수 있다.
- 예약 등록 요약에서 날짜, 테마, 시간을 확인할 수 있다.
- 예약 목록을 페이지 단위로 조회할 수 있다.
- 예약 목록의 표시 개수를 `5`, `10`, `20`개 중 선택할 수 있다.
- 이전/다음 버튼으로 예약 목록 페이지를 이동할 수 있다.
- 예약 목록에서 예약을 취소할 수 있다.
- 테마 이름, 설명, 썸네일 URL을 입력해 테마를 등록할 수 있다.
- 테마 목록에서 등록된 테마 정보를 확인하고 삭제할 수 있다.
- 시작 시간을 입력해 예약 시간을 등록할 수 있다.
- 예약 시간 목록에서 등록된 시간을 확인하고 삭제할 수 있다.
