# spring-roomescape-waiting

--- 

## 1단계

- [x] JPA를 활용하여 데이터베이스에 접근하도록 수정한다.
- [x] gradle 의존성 추가
    - [x] jpa 의존성
    - [x] lombok 의존성
    - [x] valid 의존성
- [x] 엔티티 매핑
- [x] 연관관계 매핑

## 2단계

- [x] 내 예약 목록을 조회하는 API를 구현한다.

## 3단계

- [x] 예약 대기 요청 기능을 구현한다.
    - [x] 중복 대기 신청은 불가능하다.
    - [x] 과거의 예약 대기 신청은 불가능하다.
    - [x] 예약 되지 않은 예약은 대기 신청을 할 수 없다.
- [x] 예약 대기 취소 기능도 함께 구현한다.
- [x] 내 예약 목록 조회 시 예약 대기 목록도 함께 포함한다.
- [ ] 내 예약 목록의 예약 대기 상태에 몇 번째 대기인지도 함께 표시한다.
