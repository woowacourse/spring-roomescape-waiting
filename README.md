# 방 탈출 예약 대기

## 기능 요구사항

### 예약 대기
- [x] 사용자는 예약 대기를 요청할 수 있다.
  - [x] 예약이 존재하지 않는데 예약 대기를 요청하면 예외가 발생한다.
  - [x] 과거 시간에 예약 대기를 하면 예외가 발생한다.
  - [x] 이미 같은 테마, 날짜, 시간에 대해 예약 혹은 대기 중일 시 예외가 발생한다.
- [ ] 사용자는 예약 대기를 취소할 수 있다.
  - [ ] 예약 대기 취소 시 뒷 순번들은 1 번호씩 앞당겨진다.
- [ ] 내 예약 목록 조회 시 예약 대기 목록도 함께 조회한다.
