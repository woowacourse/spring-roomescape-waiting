# 에러 코드 명세서

브라우저는 백엔드가 내려주는 `errorCode`를 기준으로 사용자 친화적인 메시지로 변환해 표시한다.  
백엔드 `message`는 에러의 의미를 설명하는 원본 메시지로 유지하고, 프론트는 화면 맥락에 맞게 문구를 다듬어 보여준다.

| 에러 코드 | HTTP 상태 | 백엔드 에러 내용 | 프론트 에러 내용 |
| --- | --- | --- | --- |
| `AUTH401_001` | 401 | 이메일 또는 비밀번호가 일치하지 않습니다. | 이메일 또는 비밀번호가 일치하지 않습니다. |
| `AUTH401_002` | 401 | 로그인이 필요한 요청입니다. | 로그인이 필요합니다. |
| `AUTH401_003` | 401 | 유효하지 않은 토큰입니다. | 인증 정보가 유효하지 않습니다. 다시 로그인해 주세요. |
| `AUTH401_004` | 401 | 토큰이 만료되었습니다. 다시 로그인해 주세요. | 세션이 만료되었습니다. 다시 로그인해 주세요. |
| `AUTH403_001` | 403 | 권한이 없습니다. | 이 작업을 수행할 권한이 없습니다. |
| `AUTH403_002` | 403 | 다른 매장에 접근권한이 없습니다. | 다른 매장의 자원에는 접근할 수 없습니다. |
| `RESERVATION404_001` | 404 | 존재하지 않는 예약입니다. | 요청한 예약을 찾을 수 없습니다. 예약 정보를 다시 확인해 주세요. |
| `RESERVATION409_001` | 409 | 이미 예약이 존재합니다. | 이미 예약된 시간입니다. 다른 시간을 선택해 주세요. |
| `RESERVATION400_001` | 400 | 지나간 날짜와 시간으로는 예약할 수 없습니다. | 지나간 날짜와 시간은 예약할 수 없습니다. 다른 날짜나 시간을 선택해 주세요. |
| `RESERVATION400_002` | 400 | 이미 지난 예약은 취소할 수 없습니다. | 이미 지난 예약은 취소할 수 없습니다. |
| `RESERVATION400_003` | 400 | 본인 예약만 변경/취소할 수 있습니다. | 본인 예약만 변경/취소할 수 있습니다. |
| `RESERVATION_TIME404_001` | 404 | 존재하지 않는 예약시간입니다. | 선택한 예약 시간을 찾을 수 없습니다. 가능 시간을 다시 조회해 주세요. |
| `RESERVATION_TIME409_001` | 409 | 예약이 존재하는 예약시간은 삭제할 수 없습니다. | 이미 예약이 존재하는 시간입니다. 다른 시간을 선택해 주세요. |
| `THEME404_001` | 404 | 존재하지 않는 테마입니다. | 선택한 테마를 찾을 수 없습니다. 테마를 다시 선택해 주세요. |
| `THEME409_001` | 409 | 예약이 존재하는 테마는 삭제할 수 없습니다. | 이미 예약이 존재하는 테마는 삭제할 수 없습니다. |
| `COMMON400_001` | 400 | 유효하지 않은 요청필드입니다. | 입력값이 올바르지 않습니다. 날짜와 예약 시간을 다시 확인해 주세요. |
| `COMMON400_002` | 400 | 경로 변수(PathVariable)가 누락됐습니다. | 요청 경로가 올바르지 않습니다. 다시 시도해 주세요. |
| `COMMON400_003` | 400 | 쿼리 스트링이 누락됐습니다. | 필요한 조회 조건이 빠졌습니다. 날짜와 테마를 다시 확인해 주세요. |
| `COMMON400_004` | 400 | 올바른 입력값 형식이 아닙니다. | 입력 형식이 올바르지 않습니다. 날짜와 시간 형식을 다시 확인해 주세요. |
| `COMMON400_005` | 400 | 올바른 쿼리 스트링 형식이 아닙니다. | 조회 조건 형식이 올바르지 않습니다. 날짜 형식을 다시 확인해 주세요. |
| `COMMON400_006` | 400 | 유효하지 않은 쿼리 스트링 값입니다. | 조회 조건 값이 올바르지 않습니다. |
| `COMMON500_001` | 500 | 예기치 못한 예외가 발생했습니다. | 문제가 발생했습니다. 잠시 후 다시 시도해 주세요. |

## 그룹별 분류

- **AUTH4xx**: 인증·인가 관련
  - **AUTH401_xxx (인증 실패)**: `001` 로그인 자체 실패, `002` 비로그인 보호된 자원 접근, `003` 토큰 위조·형식 오류, `004` 토큰 만료
  - **AUTH403_xxx (인가 실패)**: `001` role 부족 (예: 일반 사용자가 매니저 API 호출), `002` 다른 매장 자원 접근 시도
- **RESERVATION4xx**: 예약 도메인 비즈니스 규칙 위반 / 자원 부재 / 충돌
- **RESERVATION_TIME4xx**: 예약 시간 도메인
- **THEME4xx**: 테마 도메인
- **COMMON4xx**: 요청 형식·필드 검증 등 횡단 관심사
- **COMMON5xx**: 예기치 못한 서버 오류
