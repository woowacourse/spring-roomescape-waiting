# 기능 목록

- [x] 사용자는 예약된 슬롯(날짜+시간+테마)에 대한 대기를 신청할 수 있다.
    - [x] 신청한 순번대로 대기 순번이 정해진다.
    - [x] 하나의 슬롯에는 사용자당 하나의 대기 신청만 가능하다.
    - [x] 예약이 존재해야만 대기가 신청 가능하다.

- [x] 사용자는 본인의 대기를 취소할 수 있다.

- [x] 사용자는 예약 목록과 함께 대기 목록을 조회할 수 있다.
    - [x] 예약 및 대기 상태를 구분할 수 있다.
    - [x] 대기 상태면 대기 순번도 함께 조회한다.

- [] 사용자가 예약을 수정할 때
    - [] 수정 후 예약에 대기가 존재하면 해당 사용자를 대기열 마지막 순번으로 등록한다. 
    - [] 수정 전 예약에 대기가 존재하면 대기 1순위 사용자를 예약으로 승격한다.

- [] 사용자가 예약을 삭제할 때, 대기열이 존재하면 대기 1순위 사용자를 예약으로 승격한다.
