# 테마 인수 테스트 시나리오

테마 생성, 삭제, 인기 테마 조회 흐름을 검증한다.

## 시나리오 목록

| 시나리오        | 검증 내용                          |
|-------------|--------------------------------|
| 테마 목록 조회    | 생성한 테마가 목록에 포함되는지 검증           |
| 테마 삭제       | 삭제한 테마가 목록에서 제외되는지 검증          |
| 인기 테마 목록 조회 | 기간 내 예약 수 기준으로 인기 테마가 정렬되는지 검증 |

```gherkin
Feature: 테마 관리 기능

  Scenario: 테마 목록 조회
    Given 테마 생성을 요청하고
    When 테마 목록 조회를 요청하면
    Then 생성한 테마가 포함된 테마 목록을 응답받는다.

  Scenario: 테마 삭제
    Given 테마 생성을 요청하고
    When 생성한 테마 삭제를 요청하면
    Then 테마 삭제가 성공한다.
    Then 테마 목록 조회 시 삭제한 테마는 응답받지 않는다.

  Scenario: 인기 테마 목록 조회
    Given 여러 테마 생성을 요청하고
    Given 예약 시간 생성을 요청하고
    Given 기간 내 테마별 예약 수가 다르게 예약 생성을 요청하고
    Given 기간 밖 예약 생성을 요청하고
    When 인기 테마 목록 조회를 요청하면
    Then 기간 내 예약 수가 많은 순서대로 인기 테마 목록을 응답받는다.
    Then 기간 밖 예약만 있는 테마는 인기 테마 목록에 포함되지 않는다.
```
